## Hinweis zur Installation

Nidus ist noch nicht signiert. Deshalb zeigt Windows beim ersten Start eine SmartScreen-Warnung
(„Unbekannter Herausgeber"). Über **„Weitere Informationen" → „Trotzdem ausführen"** lässt sich
die Installation fortsetzen.
